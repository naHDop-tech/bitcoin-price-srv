
## Installation

```bash
$ npm install
```

## Running the app

pure
```bash
$ npm run start

$ npm run start:dev

$ npm run start:prod
```

open another terminal and run ticker service
```bash
$ npm run ticker:dev
```
## With docker
```bash
$ docker-compose up -d
```
